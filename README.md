# 🎨 Iurii Birukov - Portfolio Website

Modern portfolio website for Graphic & Motion Designer.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## 📦 Build for Production

```bash
npm run build
```

## 🌐 Deploy to Vercel

1. Push this repository to GitHub
2. Import project to Vercel
3. Deploy!

Vercel will automatically detect the configuration from `vercel.json`.

## ✨ Features

- **Hero Section** - Bold typography with personal info
- **Projects Gallery** - Showcase of selected works
- **Skills Section** - Design, Motion, and Automatizations
- **Contact Section** - Email and social media links
- **Responsive Design** - Works on all devices
- **Smooth Animations** - Modern transitions and effects

## 🛠️ Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Lucide Icons

## 📝 Customization

Edit `src/App.tsx` to customize:
- Projects array
- Skills lists
- Social media links
- Contact information
